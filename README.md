# Apetito Cafe — Official Website

Live site → [apetito-cafe.vercel.app](https://apetito-cafe.vercel.app)

A responsive single-page landing website I built for **Apetito Cafe**, a cozy cafe in Vesu, Surat known for fresh coffee, healthy wraps, and a relaxed atmosphere. Built from scratch and deployed on Vercel.

## Sections

**Hero** — Opens with the cafe name, tagline ("Fresh Coffee • Healthy Bites • Cozy Ambience"), and four quick-action buttons right up front: Call Now, WhatsApp, Order Online (Zomato), and Instagram. Makes it easy for a visitor to take action immediately without scrolling.

**About** — Short and sweet intro to what Apetito is — wraps, snacks, beverages, coffee, good vibes. Positioned as a place to relax, work, or hang out.

**Menu Highlights** — Six featured dishes with images and a "View More" link to the Zomato ordering page. Includes Mexican Rice Bowl, Orange Cheesecake, Paneer Wrap, Mediterranean Falafel Bowl, Tiramisu, and Zucchini Fries.

**Gallery ("From the eyes of Customers")** — 13 customer and interior photos in a grid layout with a hover/tap interaction. Shows the cafe interior, food, ambiance, and outside view.

**Visit Us** — Address, phone number, and opening hours (10 AM – 11 PM daily) alongside an embedded Google Maps iframe for directions.

**Floating WhatsApp Button** — A sticky WhatsApp icon sits in the corner on all sections so visitors can reach out anytime without hunting for the number.

## Built with

- **HTML5, CSS3, Vanilla JS** — no frameworks, written from scratch
- Fully **responsive** — works on mobile, tablet, and desktop
- **Open Graph meta tags** — proper link previews when shared on WhatsApp, Twitter, etc.
- SEO meta tags (title, description, keywords, author)
- Embedded **Google Maps** iframe in the contact section
- Deployed on **Vercel**

## Features

- Hero CTA buttons for Call, WhatsApp, Zomato order, and Instagram — all above the fold
- Menu cards that link directly to the Zomato ordering page
- Customer photo gallery with hover/tap to explore interaction
- Floating WhatsApp button — sticky across the whole page
- Embedded Google Maps for directions without leaving the site
- OG tags for proper social media link previews
- Clean mobile layout — designed with phone users in mind since most cafe visitors will land from Instagram or Zomato
